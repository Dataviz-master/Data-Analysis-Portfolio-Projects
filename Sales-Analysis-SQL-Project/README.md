# Sales Data Analysis Project

## Overview
A full SQL-based analysis of sales data to uncover business insights, customer behavior, and product performance. Demonstrates data cleaning, transformation, and advanced analysis skills in SQL Server.

## Key Features
- Imported and structured raw CSV data into SQL Server
- Cleaned nulls, duplicates, and standardized categorical values
- Performed advanced analysis:
  - Top selling products and revenue-generating categories
  - Cancellation and return trends
  - Customer spending behavior and age group insights
  - Peak purchase times and preferred payment modes
- Used pivot tables for gender-based product analysis
- Formatted monetary values for clear reporting

## Tech Stack
- SQL Server (T-SQL)
- Bulk Insert for data import
- Window functions, case statements, and aggregate queries
- Pivot for cross-tab analysis

## Project Structure
```
database_table.sql   # Create database & tables
data_import.sql      # Import CSV data
data_cleaning.sql    # Handle nulls, duplicates, and standardize data
analysis.sql         # All analytical queries
sales.csv            # Original dataset
README.md            # Project documentation
```
