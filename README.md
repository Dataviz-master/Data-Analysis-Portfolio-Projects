# Data Analysis Projects

This repository contains various projects related to **Excel, Power BI, and SQL** focused on **Data Analysis**.  

## Contents

- **Excel Projects:** Data cleaning, analysis, and visualization using Excel tools and formulas.  
- **Power BI Projects:** Interactive dashboards, reports, and data visualizations for business insights.  
- **SQL Projects:** Database querying, data extraction, and analysis using SQL.  

## Purpose

The projects in this repository demonstrate practical applications of data analysis techniques and tools to clean, analyze, and visualize data for actionable insights.  

## Tools & Technologies

- Microsoft Excel  
- Power BI  
- SQL (Microsoft SQL Server)  
- Data visualization and reporting tools
