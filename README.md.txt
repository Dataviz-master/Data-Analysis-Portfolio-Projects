# 🍕 Pizza Sales Analysis Project

## Overview
This project analyzes pizza sales data using **Excel** and **SQL**.  
The goal is to find sales trends, customer preferences, and revenue insights.

## Repository Structure
- **/data** → Contains the cleaned pizza sales dataset (`pizza_sales_clean.csv`)
- **/sql** → SQL scripts used for analysis
- **/excel** → Excel dashboards and pivot tables
- **/visuals** → Screenshots of the Excel dashboard

## Key Insights
- Classic pizzas contributed the highest sales revenue.
- Peak sales occur during weekends.
- Monthly revenue shows a strong upward trend.

## How to Use
1. Open `excel/dashboard.xlsx` to explore the interactive dashboard.
2. Run queries from `sql/analysis_queries.sql` to reproduce analysis.
3. Review `visuals/sales_dashboard.png` for a quick dashboard preview.
